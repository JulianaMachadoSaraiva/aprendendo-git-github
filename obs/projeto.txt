o que é um Git?
é um Software de controle de versão
ele faz o versionamento de código
ele faz isso localmente


o que é um GitHub?
plataforma
repositório remoto
rede social para devs

vc faz um projeto em conjunto



qual é a diferença entre Git e GitHub?

a diferença é que Git é local e GitHub é um repositório remoto



o link para instalação do Git é:
https://git-scm.com/download/win

obs: é importante que o nome do Git seja o mesmo nome de usuário do GitHub
o que é um repositório ou repo?
é um local onde vai ficar armazenado os arquivos do projeto
pode ser um repositório privado ou público

como criar um repositório no Git e no GitHub?
1. criar pasta no local
2. clicar com o botão direito em "git bash here" dentro da pasta
3. digitar git init
4. criar um arquivo qualquer
. abrir a página do GitGub https://github.com/
. se não possuir uma conta no GitHub, cria uma conta, clicando em sign up
. abrir a foto do seu perfil à direita superior e clicar em "seus repositórios"
. clicar no botão novo
7. dê um nome ao seu projeto
8. coloque a descrição do projeto
9. escolha se é público ou privado
10. escolher adicionar um arquivo README
11. clicar em criar repositório
12. Mude o título do projeto se desejar
13. clicar em Commit changes
14. clicar à esquerda no nome do seu "usuário/projeto"
15. você vai clonar agora o seu projeto, clicando em code
16. copie o https
17. abra o terminal do Git Bash
18. escreva "git clone e cole o endereço que vc copiou"
19. pressione enter
20. se for a primeira vez que estiver abrindo o git bash, se vc estiver no windows, ele vai perguntar se vc quer autenticar via browser e vai abrir uma aba no GitHube ele vai conseguir vincular a credencial.

PRONTO! OS REPOSITÓRIOS JÁ FORAM CRIADOS NO GIT E NO GITHUB
